# Expense Tracker

A Flutter application to track your personal expenses.

## About The Project

This project is an expense tracker app built with Flutter. It allows users to add new expenses, view a list of expenses, and see a chart of their spending. It serves as a practical example of key Flutter concepts including:
- Building user interfaces with Widgets
- Managing state
- Handling user input
- Adapting to different screen sizes and orientations
